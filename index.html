I am a UMBC student with a background in programming. I make 3D characters and animations for games using versatile software like Blender. I start with a sketch and begin a sculpt in detail stages. In Unity, I use code and animation together in pieces to create procedural animations. I break projects down to core ideas, I do the challenging parts of 3D like rigging and I don't avoid new topics. My goal is to be a technical artist or a character artist. I will create collaboratively and carry the visual storytelling on games like artists I grew up with.
