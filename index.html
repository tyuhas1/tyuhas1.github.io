I'm Trent, a UMBC Art and Computer Science student.

I make 3D characters and animations for games using versatile software like Blender.

In Unity, I use code and animation to create procedural animations. I break projects down to core ideas, do the challenging parts of 3D like rigging and I don't avoid new topics.

My goal is to be a technical artist or a 3D character artist. I like to create collaboratively and hope to carry the visual storytelling on games like the ones I grew up with.
